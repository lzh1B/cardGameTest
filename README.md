# Cocos2dx 3.17 项目配置指南

## 环境要求
- **Cocos2dx 版本**: 3.17
- **Python 版本**: 2.7.x
- **Visual Studio 2022** 必须包含以下组件：
  - `使用C++的桌面开发` 工作负载
  - 在"单个组件"中勾选：
    - MSVC v140 - VS 2015 C++ 生成工具
    - MSVC v141 - VS 2017 C++ 生成工具
    - MSVC v142 - VS 2019 C++ 生成工具
    - Windows 10 SDK (建议同时安装最新版和10.0.17763.0版本)
    - C++ ATL 和 MFC 支持
